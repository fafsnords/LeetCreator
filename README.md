# LeetCreator
### Introduction
**Leet** is a creation tool deface page for beginners or advance you can create your own deface page
### Installation
```
$ git clone https://github.com/fafsnords/LeetCreator

$ cd LeetCreator

$ pip install colorama

$ python leet.py
```
